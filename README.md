# 🚀 Task Manager Pro

A modular CLI-based Task Manager built with Python, using object-oriented design, JSON storage, and logging.

---

## 📌 Features

* Add, view, complete, and delete tasks
* Persistent storage using JSON
* Logging for actions and errors
* Input validation and error handling

---

## ▶️ How to Run

```bash
python main.py
```

---

## 🧠 Concepts Used

* OOP (Task class)
* File handling (JSON)
* Logging
* Modular structure

---

## 🛠️ Tech Stack

* Python
* JSON
* Logging
