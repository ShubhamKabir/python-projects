# 🚀 Python Mini Projects

A collection of modular Python CLI applications demonstrating core software engineering concepts such as object-oriented programming, file handling, logging, and clean architecture.

---

## 📂 Projects

### 🔹 [Task Manager Pro](./task_manager_pro)

Manage tasks with persistent storage, logging, and modular design.

### 🔹 [Log Analyzer](./log_analyzer)

Analyze log files and generate summary reports with error handling.

### 🔹 [Freelance Manager](./freelance_manager)

Track clients, projects, and earnings using structured data modeling.

### 🔹 [Expense Tracker](./expense_tracker)

Track expenses with category-wise summaries and JSON storage.
